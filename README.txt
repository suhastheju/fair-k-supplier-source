This repository contains the code for the paper "Fair clustering for data summarization: Improved
approximation algorithms and complexity insights" by Anonymous Authors. The paper is currently under
review. The code is written in Python 3.7 and uses numpy, scipy, scikitlearn and pandas python libraries.
